# renderCircle

원을 그립니다. `^GC` 명령어를 사용합니다.

```ts
function renderCircle(props: CircleCoreProps): string
```

## 사용 예시

```ts
import { renderCircle } from '@zpl-kit/zpl-core';

const zpl = renderCircle({
  diameter: 80,
  fieldOriginX: 100,
  fieldOriginY: 50,
  thickness: 2,
});
```

## Props

| 속성 | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| `diameter` | `number` | 필수 | 지름 (3~4095) |
| `fieldOriginX` | `number` | `0` | X 좌표 |
| `fieldOriginY` | `number` | `0` | Y 좌표 |
| `thickness` | `number` | `1` | 두께 (1~4095) |
| `lineColor` | `'B' \| 'W'` | `'B'` | 색상 |
