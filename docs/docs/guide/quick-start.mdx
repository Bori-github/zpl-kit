# Quick Start

zpl-kit은 두 패키지로 구성됩니다. 사용 환경에 따라 선택해 시작하세요.

import { PackageManagerTabs } from '@theme';

## @zpl-kit/react-zpl

React 환경에서 JSX로 ZPL 라벨을 선언적으로 작성합니다.

### 설치

<PackageManagerTabs
  command={{
    npm: 'npm install @zpl-kit/react-zpl',
    yarn: 'yarn add @zpl-kit/react-zpl',
    pnpm: 'pnpm add @zpl-kit/react-zpl',
  }}
/>

### 기본 사용법

```tsx
import { ZplLabel, Text } from '@zpl-kit/react-zpl';

const element = (
  <ZplLabel width={400} height={200}>
    <Text fieldOriginX={10} fieldOriginY={10}>
      Hello, ZPL!
    </Text>
  </ZplLabel>
);

const zpl = ZplLabel.print(element);
console.log(zpl);
// ^XA^PW400^LL200...^XZ
```

자세한 사용법은 [react-zpl 시작하기](/guide/react-zpl-getting-started)를 참고하세요.

## @zpl-kit/zpl-core

React 없이 Node.js / CLI 환경에서 ZPL을 생성합니다.

### 설치

<PackageManagerTabs
  command={{
    npm: 'npm install @zpl-kit/zpl-core',
    yarn: 'yarn add @zpl-kit/zpl-core',
    pnpm: 'pnpm add @zpl-kit/zpl-core',
  }}
/>

### 기본 사용법

```ts
import { renderLabel } from '@zpl-kit/zpl-core';

const zpl = renderLabel({
  type: 'label',
  props: { width: 400, height: 200 },
  children: [
    {
      type: 'text',
      props: { text: 'Hello, ZPL!', fieldOriginX: 10, fieldOriginY: 10 },
    },
  ],
});

console.log(zpl);
// ^XA^PW400^LL200...^XZ
```

자세한 사용법은 [zpl-core 시작하기](/guide/zpl-core-getting-started)를 참고하세요.
