# Introduction

ZPL(Zebra Programming Language)은 Zebra 프린터에서 라벨을 출력하기 위한 언어입니다. ZPL 명령을 직접 작성하는 방식은 반복적이고 오류가 발생하기 쉽습니다.

zpl-kit은 React 컴포넌트 또는 순수 TypeScript 함수로 라벨 구조를 선언적으로 작성하고, 이를 ZPL 문자열로 변환해 주는 도구 모음입니다.

| 패키지               | 설명                                                                  |
| -------------------- | --------------------------------------------------------------------- |
| `@zpl-kit/react-zpl` | JSX로 라벨을 선언적으로 작성하는 React 컴포넌트 라이브러리            |
| `@zpl-kit/zpl-core`  | React 없이 Node.js·서버·CLI 환경에서 사용 가능한 순수 함수 라이브러리 |

`@zpl-kit/react-zpl`을 사용하면 `@zpl-kit/zpl-core`가 자동으로 포함됩니다.
