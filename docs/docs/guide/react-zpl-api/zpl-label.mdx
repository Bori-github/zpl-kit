# ZplLabel

라벨 루트 컨테이너입니다. 모든 ZPL 요소는 `ZplLabel` 안에 선언합니다.

## 사용 예시

```tsx
import { ZplLabel, Text } from '@zpl-kit/react-zpl';

const zpl = ZplLabel.print(
  <ZplLabel
    width={400}
    height={200}
    defaultFontName="A"
    defaultFontWidth={20}
    defaultFontHeight={20}
  >
    <Text fieldOriginX={10} fieldOriginY={10}>
      Hello
    </Text>
  </ZplLabel>
);
```

## Props

| 속성                | 타입                       | 기본값   | 설명                |
| ------------------- | -------------------------- | -------- | ------------------- |
| `width`             | `number`                   | 필수     | 라벨 너비 (dots)    |
| `height`            | `number`                   | 필수     | 라벨 높이 (dots)    |
| `offsetX`           | `number`                   | `0`      | X 오프셋            |
| `offsetY`           | `number`                   | `0`      | Y 오프셋            |
| `labelOrientation`  | `'N' \| 'R' \| 'I' \| 'B'` | `'N'`    | 라벨 방향           |
| `encoding`          | `string[]`                 | `['28']` | 문자 인코딩 (UTF-8) |
| `defaultFontName`   | `string`                   | `'J'`    | 기본 폰트           |
| `defaultFontWidth`  | `number`                   | `30`     | 기본 폰트 너비      |
| `defaultFontHeight` | `number`                   | `30`     | 기본 폰트 높이      |

---

## ZplLabel.print()

JSX 트리를 ZPL 문자열로 변환하는 정적 메서드입니다.

```ts
ZplLabel.print(element: ReactElement<ZplLabelProps>): string
```

`element`는 반드시 `<ZplLabel ...>` 루트여야 합니다.
