# 설치하기

`@zpl-kit/react-zpl`은 JSX로 ZPL 라벨을 선언적으로 작성하는 React 컴포넌트 라이브러리입니다.

## 언제 사용하나요?

- React 환경에서 라벨을 생성하는 경우
- JSX 문법으로 라벨 구조를 선언적으로 작성하고 싶은 경우
- `ZplLabel.print()`로 ZPL 문자열을 간편하게 얻고 싶은 경우

## 설치

```bash
pnpm add @zpl-kit/react-zpl
```

## zpl-core 포함 여부

`@zpl-kit/react-zpl`을 설치하면 `@zpl-kit/zpl-core`가 자동으로 포함됩니다. 별도 설치가 필요하지 않습니다.
