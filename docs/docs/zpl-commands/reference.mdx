## 명령어 파라미터 설명

### 공통 파라미터

- **o (orientation)**: 방향 (N=정상, R=90도 회전, I=180도 회전, B=270도 회전)
- **h (height)**: 높이 (dots 단위)
- **w (width)**: 너비 (dots 단위)
- **x, y**: 좌표 위치 (dots 단위)
- **z**: Z 좌표 (3D 위치 지정)
- **f (field number)**: 필드 번호
- **g (gap)**: 간격
- **e (check digit)**: 체크 디지트
- **m (mode)**: 모드
- **s (security level)**: 보안 레벨
- **c (columns)**: 열 수
- **r (rows)**: 행 수
- **t (type)**: 타입
- **a, b, c, d, e, f, g, h, i, j, k, l, m, n**: 명령어별 특정 파라미터

### 폰트 크기 (Font 0)

폰트 P부터 V까지는 Font 0의 미리 정의된 크기입니다:

- **P**: 20x18
- **Q**: 28x24
- **R**: 36x30
- **S**: 40x36
- **T**: 48x42
- **U**: 60x54
- **V**: 80x72
